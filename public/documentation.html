Building My First API

The objective of this achievement is to build a server-side application.
This application in particular is for a movie application. The application
will provide users with access to information about movies, directors, and
different genres. Users will be able to create an account, update their personal
information, and create a list of their favorite movies.

Request URL HTTP Method Response
Get a list of all movies /movies GET A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET A JSON object holding data about a single movie containing the movie title, actor, genre and year released. Example: {title: Soul Food actor: Nia Long genre: Drama year: 1997}
Return data about a genre by name/title /movies/:genres GET A JSON object holding data about a single movie containing genre
Return data about a director /movies/:directors GET A JSON object holding data about a single movie director and his info. Example: {director: George Tillman Jr. birth year: 1969 bio: Short biography of director}
Allow new users to register /users POST Create JSON object with new user data. Example: {Name: Quan Mccrea User Name: atlestate1 Password: ATL1759_ Email Address: atlestate1@yahoo.com}
Allow users to update their user info(username) /users/:username PUT Update JSON object with updated username
Allow users to add a movie to their list of favorites /movies:/favorites POST A message indicating a successful add
Allow users to remove a movie from their list of favorites /movies/:favorites DELETE A message indicating a successful add
Allow existing users to deregister /users/:username DELETE Delete JSON object containing user data
